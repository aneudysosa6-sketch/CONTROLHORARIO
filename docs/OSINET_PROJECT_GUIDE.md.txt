# 1. VISIÓN DEL PROYECTO

OSINET Time ERP es un ERP profesional de Recursos Humanos desarrollado para empresas de la República Dominicana.

El objetivo del sistema es administrar completamente el ciclo de vida de un empleado desde su contratación hasta su salida de la empresa.

El proyecto está diseñado para crecer durante muchos años sin necesidad de rehacer su arquitectura.

La prioridad siempre será la estabilidad, escalabilidad y reutilización del código.

---

# 2. FILOSOFÍA

El proyecto NO es una simple aplicación Android.

Es un ERP completo.

Cada nuevo módulo debe poder funcionar posteriormente desde:

• Android

• Portal Web

• API REST

# 5. BASE DE DATOS

La base de datos local utiliza Room Database.

Todas las tablas deben tener:

Entity

DAO

Repository

ViewModel

Factory

Pantalla

Nunca acceder a Room directamente desde Compose.

Siempre utilizar Repository.

---

# 6. PATRONES

Arquitectura oficial:

MVVM

Cada Entity tendrá:

DAO

↓

Repository

↓

ViewModel

↓

Compose

Toda lógica compleja deberá implementarse mediante Engine.

---

# 7. ENGINE PATTERN

Toda lógica de negocio debe separarse de la interfaz.

Ejemplos:

PayrollEngine

PayrollPdfEngine

VacationEngine

AttendanceEngine

WhatsAppPayrollEngine

WhatsAppAttendanceEngine

DocumentEngine

NotificationEngine

Los Engine nunca deben depender de Compose.

# 12. WHATSAPP

La integración oficial será mediante n8n.

# 20. VISIÓN A LARGO PLAZO

OSINET Time ERP deberá convertirse en uno de los sistemas de Recursos Humanos más completos para la República Dominicana.

El sistema deberá competir con soluciones empresariales consolidadas, manteniendo una arquitectura moderna, modular y preparada para futuras integraciones tecnológicas.

Todo nuevo desarrollo deberá respetar esta guía como referencia oficial del proyecto.
