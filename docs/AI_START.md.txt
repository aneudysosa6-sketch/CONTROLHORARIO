# Motores (Engine)

Toda lógica compleja debe implementarse mediante Engine.

Nunca colocar lógica de negocio dentro de Compose.

---

# Room Database

Nunca acceder directamente a Room desde una pantalla Compose.

Siempre utilizar Repository.

---

# Navegación

Toda nueva pantalla debe integrarse respetando la arquitectura de navegación existente.

Nunca romper rutas anteriores.

---

# Expediente Digital

Todo documento generado por el sistema debe poder asociarse al expediente del empleado.

---

# WhatsApp

La integración oficial utiliza n8n.

La autenticación se realiza mediante el número telefónico registrado del empleado.

Nunca utilizar PIN.

Nunca permitir acceso a información de otro empleado.

---

# Seguridad

Toda información del empleado es privada.

Cada empleado únicamente puede consultar:

Su asistencia

Su nómina

Sus documentos

Sus vacaciones

Sus permisos

Sus evaluaciones

Nunca la información de otro empleado.
