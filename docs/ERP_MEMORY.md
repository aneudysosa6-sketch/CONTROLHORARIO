# Memoria Maestra del ERP

Este documento resume el conocimiento permanente del proyecto.

## Identidad

OSINET Time ERP Enterprise es un ERP móvil Android para RRHH, asistencia y nómina.

## Reglas que nunca deben olvidarse

- No borrar funcionalidades.
- No inventar clases.
- No crear placeholders.
- No reemplazar pantallas reales.
- Mantener login administrador/empleado.
- Empleado solo poncha.
- Administrador tiene acceso total.
- Código de empleado automático.
- Huella antes de ponche.
- Room local.
- MVVM.
- Compose.

## Error histórico crítico

El proyecto se dañó cuando se mezclaron:

- carpetas viejas
- carpetas nuevas
- clases duplicadas
- navegación placeholder
- módulos inventados

Por eso toda IA debe analizar antes de editar.
