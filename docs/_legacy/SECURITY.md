# Seguridad

## Roles

- ADMIN
- SUPERVISOR
- EMPLOYEE

## Administrador

Acceso total.

## Supervisor

Acceso limitado a empleados asignados.

## Empleado

Solo ponche.

## Reglas

- No exponer datos sensibles en pantallas de empleado.
- No permitir navegación manual hacia ERP desde modo empleado.
- Validar huella antes de registrar asistencia.
