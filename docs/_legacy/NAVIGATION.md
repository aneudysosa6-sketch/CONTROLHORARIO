# Flujo de navegación

## Inicio

```text
Splash / Inicio
↓
Seleccionar acceso:
- ADMINISTRADOR
- EMPLEADO
```

## Administrador

```text
Administrador
↓
Login
↓
Dashboard ERP
↓
Módulos
```

## Empleado

```text
Empleado
↓
Pantalla numérica
↓
Código empleado
↓
Identificación
↓
Huella
↓
Ponche
↓
Confirmación
```
