# UI_RULES.md

# Reglas visuales OSINET

## Estilo

- Empresarial.
- Moderno.
- Claro.
- Botones grandes.
- Contraste fuerte.
- Compatible con teléfono.

## Componentes reutilizables

Usar cuando existan:

- OSINETButton
- OSINETTextField
- OSINETCard
- OSINETHeader

## Prohibido

- Pantallas vacías.
- Placeholders.
- Textos de prueba.
- Botones sin destino.
- Navegación falsa.

## Pantalla empleado

Debe tener teclado numérico tipo llamada:

- 1 2 3
- 4 5 6
- 7 8 9
- borrar 0 aceptar

## Pantalla administrador

Debe tener acceso completo al ERP.
