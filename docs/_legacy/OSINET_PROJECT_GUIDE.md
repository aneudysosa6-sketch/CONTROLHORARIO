# OSINET_PROJECT_GUIDE.md

# Guía Maestra OSINET Time ERP Enterprise

## Objetivo empresarial

Crear una solución de control horario y RRHH para empresas que necesiten operar sin backend externo, con datos locales y arquitectura Android robusta.

## Filosofía

OSINET no es una app pequeña.  
OSINET es un ERP móvil.

Cada decisión debe favorecer:

- Escalabilidad.
- Seguridad.
- Mantenibilidad.
- Claridad.
- Estabilidad.

## Pantalla inicial oficial

La app debe abrir con:

- ADMINISTRADOR
- EMPLEADO

No debe abrir directamente en dashboard.

## Administrador

El administrador debe autenticarse y tener acceso total.

## Empleado

El empleado no entra al ERP.

Solo poncha.

## Módulos mínimos empresariales

- Empleados
- Asistencia
- Nómina
- Vacaciones
- Permisos
- Huellas
- Reportes
- Comunicación interna
- Configuración
- Seguridad
