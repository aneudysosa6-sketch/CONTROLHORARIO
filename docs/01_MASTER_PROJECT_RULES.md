# MASTER PROJECT RULES

## Reglas obligatorias
- No modificar PIN + Huella.
- Mantener lector 2Connect.
- Mantener Android + Web + Supabase.
- Mantener sincronización offline.
- No agregar funcionalidades no aprobadas.
- No eliminar funcionalidades sin autorización.
- Analizar impacto antes de cada cambio.
- Ejecutar pruebas antes de finalizar.
