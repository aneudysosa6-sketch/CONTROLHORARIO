# Módulo Empleados

## Código automático

Cada empleado debe recibir un código secuencial:

```text
00001
00002
00003
```

## Datos sugeridos

- id
- employeeCode
- nombre
- cedula
- telefono
- cargo
- departamento
- sucursal
- sueldo
- isActive
- createdAt
- updatedAt

## Reglas

No permitir códigos duplicados.  
No exigir que el usuario cree el código manualmente.
