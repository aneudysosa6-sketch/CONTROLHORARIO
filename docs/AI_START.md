# Inicio recomendado para cualquier IA

Antes de modificar el proyecto:

1. Leer documentación.
2. Analizar Workspace.
3. Detectar duplicados.
4. Detectar AppDatabase.
5. Detectar Navigation.
6. Confirmar módulos existentes.
7. Pedir autorización antes de cambios grandes.

## Prompt base

```text
Lee AGENT_RULES.md y CONTINUE_PROJECT_RULES.md.
No modifiques nada hasta analizar el proyecto actual.
No uses memoria de otro Workspace.
No inventes clases.
No crees placeholders.
```
