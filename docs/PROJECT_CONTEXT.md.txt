# Objetivo del Sistema

OSINET Time ERP es un ERP profesional de Recursos Humanos desarrollado para empresas de la República Dominicana.

El proyecto está siendo construido completamente en Android utilizando Kotlin, Jetpack Compose y Room Database.

La arquitectura está preparada para evolucionar hacia un sistema híbrido Android + Web.

# Integración WhatsApp

Actualmente se está desarrollando una integración mediante n8n.

Objetivo:

Permitir que el empleado consulte únicamente su propia información mediante WhatsApp.

La validación se realizará utilizando el número telefónico registrado en la ficha del empleado.

No se utilizará PIN.

Nunca se permitirá consultar información de otro empleado.
