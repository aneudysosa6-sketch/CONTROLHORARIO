ERRORES COMUNES

1. Room/KSP muestra error de tablas nuevas:
- Haz Build > Clean Project.
- Luego Build > Rebuild Project.

2. Base de datos local anterior no abre:
- El proyecto usa fallbackToDestructiveMigration().
- En desarrollo puede borrar la base local y crear la nueva versión.

3. No aparecen empleados al supervisor:
- Verifica que el empleado tenga departmentId correcto.
- Verifica que el supervisor tenga asignado ese departamento.

4. No aparecen eventos:
- Primero configura horario en Jornadas.
- Luego pulsa Actualizar eventos de hoy.
- Debe existir asistencia registrada para la fecha actual.

5. N8N no envía WhatsApp:
- Correcto. En este módulo queda preparado, pero la conexión real N8N queda pendiente.
