# Database Supabase

Compatibilidad objetivo: PostgreSQL 17, `pgcrypto` en `extensions`, Auth UUID, Edge Functions y Data API. Migraciones por CLI; Dashboard SQL no es fuente de verdad. Publishable key en web; `service_role` solo secreto Edge. Schemas privados no se exponen. Backups, branching local/staging y diff de drift son gates obligatorios.
