# Database Permissions

Resolución: excepción de profile > rol > denegado. Alcances: propio < departamento < sucursal < empresa < global. Capacidades existentes cubren portal, perfil, empleados, asistencia, horarios, nómina, reportes, usuarios, roles, permisos y configuración. Los códigos son contrato estable; renombrar requiere migración. Administrador obtiene catálogo activo en bootstrap. Ningún usuario puede modificar sus propios grants o alcance adicional.
